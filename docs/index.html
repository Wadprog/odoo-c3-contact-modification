Odoo 19 addon

C3 Contact Modification

A focused extension of Odoo Contacts for consistent person records, protected classification tags, and secure identity-document handling.

View repository Installation
  • Target: Odoo 19
  • Languages: English and French
  • License: LGPL-3
  • Scope: Standard Contacts app

Capabilities

Contact data that stays consistent

The module extends existing Odoo models and views. It adds no separate application or parallel contact workflow.

Required gender for people

Individual contacts select Male or Female. Companies do not display or require the field.

Consistent contact names

The person-name field guides users to enter the last name first, followed by the first name.

Protected contact tags

Twelve standard C3 tags are installed for people and companies, protected from rename, archive, and deletion.

ID document storage

Individual contacts can store one attachment-backed JPG, JPEG, or PNG identity document up to 5 MB.

Click-to-preview viewer

The full ID image loads only after Preview is clicked, then opens in Odoo's secured full-screen viewer.

English and French

User-facing labels and validation messages include French translations, including Sexe and Pièce d'identité.

Daily workflow

Use the standard Contacts form

All controls appear where contact users already work.

Create a person

Open Contacts, select Individual, enter the last name followed by the first name, choose a gender, and assign relevant C3 tags.

Classify contacts

Assign the installed tags to either people or companies: Acheteur, Démarcheur, Professeur, Auteur, Fournisseur, Employé, Contractuel, Librairie, Ecole, Administration publique, ONG, and Institution privée.

Manage an ID

For an individual contact, open the ID Document tab to upload, download, replace, delete, or click Preview. Companies do not display this tab.

Protect sensitive files

ID documents use Odoo's normal contact and attachment access checks. Preview uses the authenticated record-field route and does not create a public document URL.

Installation

Install as a standard Odoo addon

Place the repository's addon directory on the Odoo addons path, update the app list, then install or upgrade the module.

# Install odoo -d <database> -i c3_contact_modification --stop-after-init # Upgrade odoo -d <database> -u c3_contact_modification --stop-after-init # Run addon tests with this repository's Docker setup docker compose run --rm web odoo -d c3_contact_test \ -i c3_contact_modification \ --test-enable \ --test-tags /c3_contact_modification \ --stop-after-init \ --without-demo=all
